Real-Time Collaborative Document Editor

A Google Docs–style real-time collaborative editor that allows multiple users to edit the same document simultaneously. The platform uses WebSockets for instant synchronization, enabling live updates across all connected clients.

This project demonstrates real-time systems, WebSocket communication, and distributed backend architecture.

Features

Real-time collaborative document editing

WebSocket-based synchronization

Automatic document creation and autosave

Persistent document storage in MongoDB

Document room system for multi-user editing

REST API for document management

Modular backend architecture
